# ダウンロードファイル格納ディレクトリ

日次で生成されるCSVファイルをここに配置する。
Netlifyデプロイ時に `/data/` パスで公開される。

## ファイル命名規則

- 日次: `nta_new_YYYY-MM-DD.csv`
- 週次: `nta_new_week_YYYY-WXX.csv`
- 月次: `nta_new_YYYY-MM.csv`

## DL URL例

- `https://list.kym-work.com/data/nta_new_2026-05-22.csv`
- `https://list.kym-work.com/data/nta_new_2026-05.csv`
